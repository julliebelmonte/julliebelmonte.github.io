Mulheres na Ciência de Dados

Análise de Dados Longitudinais State Of Data

Este estudo analisa a evolução da presença feminina no setor no Brasil com base nos dados das pesquisas State of Data Brazil entre 2021 e 2023. A investigação considera variáveis como distribuição salarial, senioridade, experiência e ocupação de liderança. Os resultados apontam avanços graduais da participação feminina, mas destacam desigualdades persistentes, especialmente no acesso a posições estratégicas.
